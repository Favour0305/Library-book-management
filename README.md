Library Book Management System

A simple Python program to manage a library of books. Users can add, view, search, and remove books. Developed following the Software Development Life Cycle (SDLC) principles.


---

Features

Add a book (Title, Author, Year)

View all books

Search by title

Remove a book

Easy-to-use text-based menu



---

How to Run

1. Make sure Python 3 is installed.


2. Run the program:



python library_book_management_system.py

3. Follow the menu to perform actions.




---

SDLC Overview

Requirement Analysis: Manage books (add/view/search/remove)

System Design: Python list of dictionaries, modular functions

Implementation: Functions match design nomenclature

Testing: Input validation, search and delete tested

Deployment: Runs locally in Python 3

Maintenance: Can be extended with file storage, GUI, or borrowing system
